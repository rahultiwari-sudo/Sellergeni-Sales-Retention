3 · Monthly Run Rate (₹) — MRR = Σ (Deal ÷ Closed Months)
Each deal contributes Deal Amount ÷ Closed Months recurring revenue per month. MRR for a month = sum of run-rate from all deals active in that month.

How MRR is Calculated
📌 Deal MRR = Deal Amount ÷ Closed Months
📌 A deal closed for 3 months contributes its run-rate to each of those 3 months
📌 Actual MRR = Latest month's own MRR + overlapping run-rate from prior months still active
Example: Apr deal ₹30K/3mo = ₹10K/mo. If May has its own ₹15K MRR + April's ₹10K still running → Actual MRR = ₹25K
